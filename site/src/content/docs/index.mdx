## Why Hive?

AI coding assistants load context statically. A typical `CLAUDE.md` grows to 800+ lines. Most is irrelevant to the current task. Every session pays the full token cost.

Hive replaces static context with **on-demand vault queries** via the [Model Context Protocol](https://modelcontextprotocol.io/). **67–82% token reduction** on targeted queries.

## Quick Install

**Claude Code:**
```bash
claude mcp add hive -- uvx hive-vault
```

**Other clients:** point your MCP config at `uvx hive-vault` via stdio transport.

No cloning, no venv, no setup. [Learn more →](/hive/getting-started/)
